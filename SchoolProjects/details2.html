Java Arcade Game

Project

Create an old arcade style game from scratch in Eclipse using standard Java libraries.

Final Product

An auto-scrolling platformer where the player tries to avoid and shoot Java icons while trying to stay on the moving platforms. After all enemies in a group are destroyed, the level and speed of the enemies increases. Game continues until the players health is reduced to 0, or the player falls off the moving platforms, after which the user inputs a name and their score gets added to an SQLite database.

Learning

Greatly reinforced object-oriented programming principles, and learned how to incorporate an SQLite database into a java project.
